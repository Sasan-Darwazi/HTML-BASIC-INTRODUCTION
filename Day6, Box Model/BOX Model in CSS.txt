BOX Model in CSS:
.Every element in CSS can be represented using the BOX model
.It helps the developer to develop and manipulate the element
.it consists of 4 edges:
 .Content edge -  it comprises of the actual content
 .Padding Edge - it lies in between content and border edge
 .Border  Edge - The border edge follows padding
 .Margin edge -  it is outside the border and controls the margin of the element

Examples:
 #Style{
border:2px solid red;
margin: 5px;
padding: 20px;
width: 20px;
height: 20px;
